Projects

Mooncake

This is the web application built during my Code+ internship during Summer 2020 that streamlines the course permission number process at Duke for students and faculty.

Read More

Built using Ruby on Rails, HTML/CSS, JavaScript, and MySQL, this web application allows students to request for course permission numbers from faculty members at Duke University on one platform. Additionally, faculty members are able to manage all the requests they receive from students and grant permission numbers on the platform without having to use other tools such as Excel or Qualtrics. If you have Duke NetID authentication, you can access the app at mooncake.colab.duke.edu

Design POD Tool Finder

A responsive website that helps engineering students locate tools within the Duke Engineering Design POD.

Read More
